About Scannova

A cybersecurity project built during my internship to learn real world security concepts

What is Scannova?

Scannova is a free online file malware scanner that allows anyone to upload a file, APK, or document and scan it against 70+ antivirus engines using the VirusTotal API. It was built as my internship submission project to understand how real cybersecurity tools work in the real world.

Tech Stack Used

HTML5
CSS3
JavaScript
Python Flask
VirusTotal API
jsPDF

Honest Note 💚

I am a beginner in web development and cybersecurity. I have basic knowledge of HTML, CSS, and JavaScript from my studies. For this project I took help from AI to understand concepts, write code, and learn how things connect together.


This project helped me understand how real cybersecurity tools work, how APIs are used in web applications, how frontend and backend communicate, and how to deploy a live website. I learned a lot while building Scannova and I am proud of what I created with the help of AI as my learning guide! 😊
